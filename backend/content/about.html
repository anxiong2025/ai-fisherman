关于 AI Fisherman

AI Fisherman 是一个专注于 AI 技术学习和实践的平台。我们提供关于以下主题的高质量内容：

AI Agent 开发

学习如何构建智能代理（AI Agent），包括任务规划、工具调用、记忆管理等核心概念。我们提供从入门到进阶的完整学习路径。

RAG 系统

RAG（Retrieval-Augmented Generation）是将检索和生成结合的技术。我们教你如何构建高效的 RAG 系统，包括向量数据库选型、Embedding 模型选择、检索策略优化等。

LLM API 使用

掌握主流大语言模型 API 的使用，包括 OpenAI GPT、Anthropic Claude、Google Gemini 等。学习 Prompt Engineering、Function Calling、Streaming 等高级技巧。

Prompt Engineering

提示工程是与大模型高效沟通的艺术。我们分享实用的 Prompt 技巧，帮助你获得更好的 AI 输出效果。
